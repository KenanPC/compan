Compandroid Token QR

Offline QR generator for a read-only GitHub token. The token stays in this page and is erased automatically.

Use only the installed app token here: Metadata read, Contents read, and Actions read. Do not paste an LLM agent write token into this page.

  • Works from a local file with no network calls.
  • Clears the token after 2 minutes of inactivity.
  • Clears immediately when you click Erase Session or close the page.
